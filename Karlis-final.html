Below we used parameter family='binomial' because of our bernoulli response variable-survival. Now, in order to screen variables we based our methodology on https://statweb.stanford.edu/~tibs/ftp/strong.pdf. Based on the paper, we can use a methodology called sequential strong rules that pinpoints variables that can be safely removed over a grid of tuning parameter values. Since during lasso methodology we have already computed the estimated parameters for each $\lambda$, the sequential strong rule discards the $j_{th}$ predictor from the optimization problem at $\lambda_k$ if $$|x_j^T(y-X\hat{\beta}(\lambda_k -1))|< 2 \lambda_k -\lambda_{k-1}.$$

Now we need to remember that for each of the hypothesis tests, we end up making a comparison between the mean difference of the 2 groups of the randomized samples and the initial difference before starting randomization. Depending on this, we end up rejecting the null hypothesis of the insignificance of the gene on survival or not. In order to decide for the rejection we need a threshold, above which we will reject the null hypothesis and declare the gene significant. We start by picking delta-threshold randomly.

In [28]:
delta=2.5
samr.plot(samr.obj,delta)

The above picture rejects no null hypothesis and the choice was completely at random, having no idea what the FDR score is. In contrast, we will pick from delta table the delta that corresponds to the minimum FDR (0.02905757). We also don't want to send FDR to zero because that would reject the significance of nearly all genes.
