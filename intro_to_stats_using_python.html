Regression¶

  • We use regression when we want to examine the relationship between one variable (the dependent variable) and one or more other variables (the independent variables).

  • In particular we want to find out whether the dependent variable really depends on the independent variables.

  • In the simplest case, we have just two variables (the dependent and one independent), which we handle with simple linear regression analysis.

  • Linear regression assumes that the relationship between the dependent variable $y$ and the independent variable $x$ is of the form:

$$y = \alpha + \beta x + \epsilon$$
  • We have a set of $y_i$ and $x_i$ from our observations, we want to estimate $\alpha$ and $\beta$.

  • The vertical lines are pointwise prediction intervals.

  • The prediction intervals show the range that will include with 95% probability the $y$ value for a given $x$ value.

  • It is also useful to plot the regression confidence interval. This is the 95% confidence interval for the regression line.

  • It is different from the prediction confidence interval, which is the 95% confidence interval for our predictions.

  • We can see them both on one figure.

  • To create the figure we have to do some work.

  • First, we have to make sure that the data do not contain NaNs and are sorted on the $x$ axis.

  • Then, we must extract the regression confidence intervals and the prediction confidence intervals from a data structure returned by the summary_table function.

  • In particular, the regression confidence intervals are columns 4, 5 and the prediction confidence intervals are columns 6, 7 of the second return value of the summary_table function.

  • Admittedly, this is ugly; but it does gives us some extra info on what regression really does.

$R^2$¶

  • What is $R^2$?

  • If $R^2 = 1$ then the all the $y_i$ fall into the regression line; the line fits the data perfectly.

  • If $R^2 = 0$ then the regression line does not fit the data at all.

  • Let's see an example.

  • We will create an artificial data set, where the values of $y$ are derived from $x$ with some noise added to them.

  • The noise will be larger towards the two ends of the interval of $x$ values, smaller around the middle of the $x$ interval.

  • We denote by $\bar{y}$ the mean of the observed data:
$$\bar{y}=\frac{1}{n}\sum_{i=1}^n y_i$$
  • By $SS_\text{tot}$ the total sum of squares:
$$SS_\text{tot}=\sum_i (y_i-\bar{y})^2$$
  • By $SS_\text{res}$ the residuals sum of squares:
$$ SS_\text{res}=\sum_i (y_i - f_i)^2 $$
  • Then $R^2$ is the ratio of the explained variance to the total variance. The explained variance is $SS_\text{res}/n$ and the total variance is $SS_\text{tot} / n$, so:
$$R^2 = \frac{SS_\text{res}/n} {SS_\text{tot}/n} $$
  • For example, if $R^2 = 0.49$, it means that our regression model explains 49% of the variability between the two variables. The other 51% remains unaccounted for.

Logistic Regression¶

  • When the response variable is binary, we cannot use ordinary regression.

  • That is because ordinary regression assumes that the response is a continuous variable.

  • We can use instead logistic regression.

  • In logistic regression we want to model the conditional probability $P(Y = 1 | X = x)$.

  • A convenient transformation is the logit or logistic transformation:
$$ log\frac{p}{1 - p} $$
  • That means that the logistic regression model is:
$$ log\frac{p}{1 - p} = \beta_0 + \beta_1 x$$
  • And we have, solving for $p(x)$:
$$ p(x) = \frac {1}{1+e^{-(\beta_0 + \beta_1 x)}} $$
  • We can see that this function looks like an S.
